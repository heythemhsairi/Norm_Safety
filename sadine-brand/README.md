# Sadine Tours — Brand Identity System

A complete brand book for Sadine Tours: a Tunisian travel agency repositioning into a premium, digital-first travel house.

## Files

- `index.html` — the visual brand book. 26 cinematic slides, sized 16:9 for screen and laid out for print-to-PDF (`@page 1440px 810px`). Each slide is one printed page.
- `strategy.md` — the written brand strategy: positioning, mission, vision, values, personality, tone of voice, audience archetypes, all design rules, and the closing standard.
- `assets/sadine-full.svg` — primary horizontal lockup (white wordmark + gold bird).
- `assets/sadine-icon.svg` — bird-only icon.

## How to view

Open `index.html` directly in a browser. Best at 1440 px viewport width or wider.

## How to export to PDF

1. Open `index.html` in Chrome.
2. **Print** → **Save as PDF**.
3. Layout: **Landscape**. Margins: **None**. Background graphics: **On**. Scale: **Default / 100%**.
4. Each slide prints as one page (page size matches slide).

## How to share

The book is one self-contained HTML file plus the SVG assets. To send: zip the `sadine-brand/` folder, or deploy the folder to a static host.

## Fonts

Loaded from Fontshare and Google Fonts at runtime — no install required:

- **Satoshi** (Fontshare) — display
- **Manrope** (Google Fonts) — text & UI
- **Alexandria** (Google Fonts) — Arabic display
- **Cairo** (Google Fonts) — Arabic UI fallback

## Priority order if budget is tight

1. **Logo PNG/EPS exports** + **favicon set** — needed for any web/print launch — €0 (in-house)
2. **Boarding pass + welcome envelope + wax seal** — the *signature artifact* that becomes the brand's calling card — €600–1200 for first run of 200
3. **6 hero photos** — Istanbul / Marrakech etc. — license from premium stock (Stocksy, Trunk Archive) for €800–2000, or commission a 2-day shoot for €2500–4000
4. **Web hero video** — 6-second loop — €600–1500 from a freelancer or a single licensed clip
5. **Business cards + lapel pins** — €300–600 for first run of 100 cards + 20 pins
6. **Vehicle wrap + storefront signage** — only when those exist physically

Everything else can wait until growth justifies it.
